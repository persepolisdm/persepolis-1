شروع به کار با پرسپولیس

در اینجا راهنمای نصب پرسپولیس روی سیستم‌عامل‌های مختلف را می‌بینید:

گنو/لینوکس


اوبونتو و دیگر توزیع‌های بر مبنای دبیان:
کافیست فقط دستورات زیر را در ترمینال وارد کنید: 
sudo add-apt-repository ppa:persepolis/ppa
sudo apt update
sudo apt install persepolis

آرچ‌لینوکس و دیگر توزیع‌های بر مبنای آرچ:
نصب آخرین انتشار (پیشنهاد می‌شود): 
sudo pacman -S persepolis
نصب با آخرین تغییرات از گیت‌هاب: 
yay -S persepolis-git

فدورا:
sudo dnf install persepolis

اپن سوزه:

برای نصب پرسپولیس بر روی openSUSE ویکی را دنبال کنید

مک

برای نصب پرسپولیس روی مک از صفحه انتشار نسخه مک را دانلود کرده و در پوشه Applications قرار دهید

اطلاعات بیشتر در ویکی پرسپولیس


ویندوز

فایل نصاب ویندوز را با توجه به معماری سیستم خود از صفحه انتشار دانلود کرده و اجرا کنید.
